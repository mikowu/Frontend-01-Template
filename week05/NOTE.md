
#### HTTP协议
##### HTTP 是纯粹的文本协议，它是规定了使用 TCP 协议来传输文本格式的一个应用层协议。
##### 通讯是由浏览器端首先发起的
- request：
  - request line
    - method：GET、POST、HEAD、PUT、DELETE、CONNECT、OPTIONS、TRACE
    - path
    - version
  - head
  - body
 - response：
  - response line
    - status code
    - status text
    - version
  - head
  - body
